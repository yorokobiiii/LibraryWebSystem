# 📚 LibraryWebSystem

Современная веб-система для управления библиотечным каталогом с гибким поиском.  
Реализована на платформе `.NET 8` с использованием `Blazor Server`, `Entity Framework Core` и `Docker`.

---

##  Возможности

- 🔍 **Гибкий поиск** по названию и автору с игнорированием регистра и частичным совпадением
- 🎛 **Фильтрация** по диапазону годов издания и формату файла (`PDF`, `EPUB`, `FB2`)
- 📖 **Каталог книг** в адаптивной сетке с карточками и автоматическим объединением авторов
- ➕ **Добавление книг** в базу 
- 🐳 **Полная контейнеризация** с multi-stage сборкой и сохранением данных через Docker Volume
- 🔌 **REST API** для получения, создания и просмотра конфигурации системы

---

## 🛠 Стек технологий

| Технология | Назначение |
|------------|------------|
| `.NET 8 / ASP.NET Core` | Кроссплатформенная среда и веб-фреймворк |
| `Blazor Server` | Интерактивный интерфейс на C# (SignalR) |
| `Entity Framework Core` | ORM, подход CodeFirst, миграции |
| `SQLite` | Легковесная файловая СУБД |
| `Docker & Docker Compose` | Изоляция, сборка и запуск окружения |
| `DataAnnotations` | Клиентская и серверная валидация моделей |
